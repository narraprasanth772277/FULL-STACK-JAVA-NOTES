Modules In Java
================
There are three modules in java.

				JAVA

	JSE/J2SE		JEE/J2EE		JME/J2ME

			> N-tier App 



> Standalone App 
-----------------
A normal java program which contains main method is called standalone application.
ex:
	class  Test
	{
		public static void main(String[] args)
		{
			-
			-
			-
		}
	}

> Desktop App 
------------
It is a software application which is used to perform perticular task.
ex:
	Control panel
	Recycle bin 
	VLC Media player 
	and etc.

> Two-tier App 
--------------
Having more then one tier is called two tier application.

Diagram: java2.1
Diagram: java2.2


> Distributed App 
-------------------
In client-server application, if multiple clients are given the request to main server then main server will distribute those request to it's parallel servers to reduce the burdun of main server such type of application is called distributed application.

Diagram:java2.3

> Enterprises App
------------------
An application which deals with large business complex logic with the help of middleware services is called enterprise application.

Here middle ware services means authentication,autherization, malware production, firewall ,security and etc.

ex:
	Facebook 
	Online shopping websites	

> ERP App 
------------
ERP stands for Enterprise Resource Planning.

It is used to maintain the data in a enterprise.

Diagram: java2.4


> N-tier App 
-------------
Having more then two tier is called N-tier application.

Diagram: java2.5

> Mobile App
-----------
It is a software application or a program which is developed for wireless network devices like phone,tab,cell,cellular and etc rather then laptop's and pc's.
ex:
	GPay
	PhonePay
	templeRun 
	and etc.
